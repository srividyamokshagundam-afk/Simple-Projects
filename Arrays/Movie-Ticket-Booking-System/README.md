# Movie Ticket Booking System

A console-based Movie Ticket Booking System developed using Java and Object-Oriented Programming (OOP).

## Features
- Display available movies
- Book movie tickets
- Cancel booked tickets
- Generate ticket bill

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner

## Concepts Used
- Classes and Objects
- Constructors
- Methods
- Loops
- Conditional Statements
